# Commercial Zoning Scraper

This Actor scrapes building permit data from local government portals. It is designed to monitor new commercial, industrial, and mixed-use building permits and provide structured output.

## Features:
- Scrapes local government websites for zoning updates and building permits.
- Filters results based on zoning area (optional).
- Collects data such as permit type, zone, address, and issue date.

## Usage:
1. Define the start URLs for the portals you want to scrape.
2. If needed, filter by zoning areas.
3. Run the Actor to collect data in the selected output format.
